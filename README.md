MERN Stack E-Commerce Web Application

A full stack **E-Commerce Web Application** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.
The application allows users to browse products, add items to a cart, place orders, and manage products through an admin dashboard.

---

Features

* User authentication with JWT
* Product listing and product search
* Product details page
* Add to cart functionality
* Checkout and order management
* Product reviews and ratings
* Admin dashboard for product management
* RESTful API backend
* Responsive UI

---

Tech Stack

Frontend

* React.js
* Redux
* React Bootstrap
* Axios

Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JSON Web Token (JWT)

⚙️Installation & Setup

1️⃣ Clone the repository

```
git clone https://github.com/yourusername/mern-ecommerce-store.git
```

2️⃣ Install dependencies

Backend:
cd server
npm install


Frontend:
cd client
npm install


3️⃣ Run the application

Backend:
npm run server


Frontend:
npm start
